Basic Directives

디렉티브(Directive)
디렉티브는 v- 접두사가 있는 특수 속성이다. 디렉티브 속성 값은 단일 JavaScript 표현식이 된다 (v-for는 예외). 디렉티브의 역할은 표현식의 값이 변경될 때 사이드 이펙트를 반응적으로 DOM에 적용한다.

template
template : 여러 개의 태그들을 묶어서 처리해야 할 경우에 template를 사용한다. v-if, v-for, component등과 함께 많이 사용한다.

v-text
v-text는 자바스크립트의 innerText와 같은 역할을 한다. html 태그가 적용되지 않고 문자열이 그대로 보여진다.

v-bind
v-bind는 엘리먼트의 속성과 바인딩 처리를 위해서 사용한다. 약어로 ":"로 사용이 가능하다.

v-html
v-html은 자바스크립트의 innerHTML과 같은 역할을 한다. html 태그가 적용된 화면이 보여진다. 이중 중괄호(mustaches)는 HTML이 아닌 일반 텍스트로 데이터를 해석한다. 실제 HTML을 출력하려면 v-html 디렉티브를 사용한다.

v-show
v-show는 조건에 따라 엘리먼트를 화면에 렌더링 한다. style의 display를 변경한다.
